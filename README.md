## 🖼️ Logo Setup

### Option 1: Use SVG Logo (Already Added)
The SVG logo is already in `images/logo.svg` and works great on all devices.

### Option 2: Add PNG Logo
1. Save your PNG logo as `logo.png`
2. Place it in the `images/` folder
3. Update `index.html` line 28 to use PNG:
```html
<img src="images/logo.png" alt="Corbett Cab Riders Logo" class="logo">
```
